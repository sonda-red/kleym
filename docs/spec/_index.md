Kleym keeps separate specs for the operator and CLI so CLI work can reference
the CLI contract without changing the operator contract by accident.

| Spec | Scope |
| --- | --- |
| [Operator Spec](operator) | Authoritative product, API, and reconciliation behavior for `kleym-operator`. |
| [CLI Spec](cli) | Read-only inspection CLI contract for `kleym`. |
